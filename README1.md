# 🔐 AES-128 Visualizer (Tkinter)

An educational Python application that demonstrates the **AES-128 encryption algorithm step by step**, including both encryption rounds and key expansion.

---

## 📌 Features

* 🔍 Step-by-step AES encryption visualization
* 🔑 Full Key Schedule (Key Expansion) explanation
* 🎨 Highlight changes between steps
* 🖥️ Simple graphical interface using Tkinter
* 📚 Designed for students learning cryptography

---

## 🧠 What is AES?

AES (Advanced Encryption Standard) is a symmetric encryption algorithm used worldwide to secure data.

This application helps you understand:

* SubBytes
* ShiftRows
* MixColumns
* AddRoundKey
* Key Expansion

---

## 🚀 How to Run

### 1. Install Python

Make sure you have Python installed (version 3.x)

### 2. Run the program

```bash
python aes_visualizer_tk.py
```
